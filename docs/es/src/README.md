# Documentación de Vallumix

Bienvenido a la documentación de Vallumix. Vallumix es un motor modular de endurecimiento de Linux con perfiles alineados con CIS, escrito en Rust. Automatiza el endurecimiento de servidores aplicando controles alineados con los benchmarks CIS, permitiéndote seleccionar un perfil basado en el rol de tu servidor y generar informes detallados de cumplimiento.

Esta documentación cubre la instalación, uso, configuración y directrices para contribuir.
